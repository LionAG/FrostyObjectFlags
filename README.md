# FrostyObjectFlags
An object flags calculator plugin for [Frosty Editor](https://www.frostytoolsuite.com/downloads.html).

### How to install

1. Use Frosty Editor v1.0.6.1 or higher.
2. Download the dll and copy into the *Plugins* directory.
